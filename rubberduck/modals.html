Bootstrap Modal Cheatsheet

An introduction to building modals in bootstrap

What are Modals

Bootstrap Modals offer a lightweight, multi-purpose JavaScript popup that’s customizable and responsive. They can be used to display alert popups, videos, and images in a website. Bootstrap-based websites can use Bootstrap modals to showcase, for example, terms and conditions (as part of a signup process), videos (similar to a standard light box), or even social media widgets. You don’t have to write a single line of JavaScript to use them! All the code and styles are predefined by Bootstrap. All that’s required is that you use the proper markup and attributes and they just work.

Components

Bootstrap Modals are divided into three primary sections: the header, body, and footer. Always include in your modal headers dismiss actions whenever possible, or provide another explicit dismiss action.

Examples

Basic Modal

The code below creates a basic modal.


<!-- Create the Button to trigger modal -->
        <button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
        Button Name Here
</button>

<!-- Create Modal Structure-->
<div class="modal fade" id="YourModal" tabindex="-1" aria-labelledby="YourModalLabel" aria-hidden="true">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class=

      <!-- Create Modal Header-->
      "modal-header">
        <h5 class="modal-title" id="YourModalLabel">Your Modal title</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal" aria-label="Close"></button>
      </div>
      <div class=

       <!-- Create Modal Body-->
      "modal-body">
        Your modal content here.
      </div>
      <div class=
      
       <!-- Create Modal Footer-->
      "modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Close</button>
        <button type="button" class="btn btn-primary">Save changes</button>
      </div>
    </div>
  </div>
</div>

Click the button to open an example of the basic modal.

Fullscreen Modal Example

To create a modal that covers the entire viewport of a user's screen add the default tag .modal-fullscreen .

The table below shows tags to style the modal as fullscreen at these break points. The default Bootstrap modal is medium sized modal.
Size Class Modal max-width
Small .modal-sm 300px
Default None 500px
Large .modal-lg 800px
Extra large .modal-xl 1140px
