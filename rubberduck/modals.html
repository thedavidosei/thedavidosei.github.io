Bootstrap Modal Cheatsheet

An introduction to building modals in bootstrap

What are Modals

Modals are pop-up windows that can appear when a page loads or when triggered by a button. Modals are used for alert pop-ups, mailing list sign-ups, welcomes messages and log-in prompts.

How to build and display a Modal

A modal is composed of 3 main parts: Header, Body & Footer. The Header is where the title and close button are placed. In the body there can be text, images or videos. In the footer there should be an action button like "Sign up!" You will need to use HTML, CSS & JavaScript. There are lots of variations of the basic modal and they can be viewed in the Bootstrap 5 documentation here.

Basic Modal


The code below creates a basic modal.


Step 1

Start by coding the trigger button to display your modal


       < div >< button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#myModal" >
        Open modal
Step 2

Create your modal structure by adding two divs with the class .modal-dialog and .modal-content


       < div class="modal fade" id="myModal" >
      < div class="modal-dialog" >
        < div class="modal-content" >
Step 3

Add the modal header by opening a div with the class .modal-header and a heading tag with the class .modal-title and then add an icon to dismiss the modal in the header


          < div class="modal-header" >
            < h4 class="modal-title">Modal Heading 
            < button type="button" class="btn-close" data-bs-dismiss="modal" > < /button > < /div >
Step 4

Create a div with the class .modal-body and add the relevant text to the modal 


          < div class="modal-body" >
            Yay! You created a Modal
Step 5

Create a div with the class .modal-footer and add an action button for the user. 

          < div class="modal-footer" >
            < button type="button" class="btn btn-success" data-bs-dismiss="modal" > Save Changes

Pro-tip

Dont forget to close your divs or else the modals wont work.
